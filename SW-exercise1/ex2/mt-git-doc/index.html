"学習したGitのコマンド"

設定

    ユーザー名の設定 $ git config --global user.name "ユーザ名"
Emailアドレスの設定 $ git config --global user.email "Emailアドレス" 設定内容の表示 $ git config --list

リポジトリの作成

リポジトリを作成し、作業ディレクトリをGitの管理下に置く $ git init

コミットの作成

ファイルをリポジトリへの登録対象に追加する $ git add リポジトリへの登録対象ファイルの変更をリポジトリに記録する $ git commit

状態の確認

リポジトリの状態を確認する $ git status コミットの履歴を確認する $ git log 変更差分を確認する $ git diff
